# Development

## Setting Up the Development Environment

1. **Clone the repository:**

    ```bash
    git clone https://github.com/<your-username>/lance-ray.git
    cd lance-ray
    ```

2. **Install dependencies:**

    ```bash
    pip install -e .[dev]
    ```

## Running Tests

Run all tests using [pytest](https://docs.pytest.org/):

```bash
pytest
